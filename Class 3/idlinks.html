Introduction

If we add the id attribute to a tag, we can link directly to that element by using a relative url that includes the pound sign (#) followed by the element id.



Element IDs

Add an id attribute to any element. If you want to link to that element, make sure the id is unique!

<h2 id="element_ids"> Element IDs</h2>


Linking to an element

Use a relative URL (no protocol or server name). You don't need to even add a file name, just the id preceeded by #
